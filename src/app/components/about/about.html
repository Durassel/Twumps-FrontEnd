Topic

Our objective is to carry out an unconventional and creative project that is out of the ordinary. We found many ideas but only one stood out from the crowd : create some interesting statistics about Donald Trump.


How can we do that ?

Donald Trump loves to use social media to communicate about his projects and beliefs. Especially Twitter that he uses very frequently. It is a simple way to express short ideas.

Our solution is to use Donald Trump's Twitter account to make various statistics in order to learn more about him and his behaviour. Thanks to Twitter, we can take his tweets and use them for analysis. For example, it’s possible to count how many times one word is used, or find the lexical field employed by Donald Trump.


How can we get Donald Trump's tweets ?

Twitter has got an API that allows to make requests in order to retrieve some data. The problem is that we can only make 3200 requests per month. That is a huge problem for us, because Donald Trump has twitted more than 35 000 tweets.


What is the solution ?

To solve this problem, we used a Python script to browse all Donald Trump's tweets (Twitter scrapping), i.e. it automatically reads Donald Trump's profile, in order to retrieve the data. This method allowed us to create a first database.

Tweets information about the writter was not very accurate, so our analysis could be incorrect. That's why we contact Ben SUGERMAN who created this website : didtrumptweetit.com. It distinguishes tweets written by Donald Trump or his team, in the form of a percentage of accuracy. He shared his database with us, which we merged with the one we had.

Graphics

We have chosen to create different and multiple graphics about Trump's tweets, every of them show something different

Words cloud

By using a words cloud, we wanted to show you how much Trump use his words.To make it concreate, we take Trump's tweets words from our database and we delete the linking words before to analyse and count every found word. After that we select the most used words and show them in our cloud. Bigger a word is, more it is used by Trump


Emotion

Are not you curious to know if Trump's tweets a politically correct? You can get an answer with our emotion graphics. Thanks to the sentiment library we are able to give a mark to each word and then count it for a whole tweet to know if it is positive, neutral or negative. It takes Trump's tweets into our database and count to give a mark for each year, plus a global one that we are able to show in our graphic


Timeline

Trump and history. With our timeline you are able to watch through the time the most retweeted Trump's tweets along with some Trump important days. To get it, we have taken tweets from our database and selected the most retweeted tweets. Then we order them and put them into our timeline


Country quoted

Which countries Trump talks about? Obviously mostly United States of America. But not only. And with this graphic we let you know which countries all around the world Trump talks about. try to find your country! By using data from Trump's tweets inside our database and by isolating countries quoted in each tweet and after counting each occurrence, we are able to create these graphics and show it to you
