Team

We are students from Efrei Paris (Engineering School of Information and Digital Technologies). We are studying Software Engineering. We are currently in the last year of the engineering cycle. And this is the result of our project of "Event-driven Asynchronous Programming".

Topic

Our objective is to carry out an unconventional and creative project that is out of the ordinary. We found many ideas but only one stood out from the crowd : create some interesting statistics about Donald Trump.

How can we do that ?

Donald Trump loves to use social media to communicate about his projects and beliefs. Especially Twitter that he uses very frequently. It is a simple way to express short ideas.

Our solution is to use Donald Trump's Twitter account to make various statistics in order to learn more about him and his behaviour. Thanks to Twitter, we can take his tweets and use them for analysis. For example, itâ€™s possible to count how many times one word is used, or find the lexical field employed by Donald Trump.

How can we get Donald Trump's tweets ?

Twitter has got an API that allows to make requests in order to retrieve some data. The problem is that we can only make 3200 requests per month. That is a huge problem for us, because Donald Trump has twitted more than 35 000 tweets.

What is the solution ?

To solve this problem, we used a Python script to browse all Donald Trump's tweets (Twitter scrapping), i.e. it automatically reads Donald Trump's profile, in order to retrieve the data. This method allowed us to create a first database.

Tweets information about the writter was not very accurate, so our analysis could be incorrect. That's why we contact Ben SUGERMAN who created this website : didtrumptweetit.com. It distinguishes tweets written by Donald Trump or his team, in the form of a percentage of accuracy. He shared his database with us, which we merged with the one we had.
